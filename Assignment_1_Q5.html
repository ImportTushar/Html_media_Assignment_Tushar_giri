DOCTYPE

In HTML (Hypertext Markup Language),
a DOCTYPE declaration, often referred to as a DOCTYPE,
it is an instruction that specifies the version of HTML in which a web page is written.
It helps web browsers and other user agents to interpret and render the web page correctly,
as different versions of HTML may have different rules and features.
A typical DOCTYPE declaration is placed at the very beginning of an HTML document,
usually right before the opening < html > tag.
It consists of the < !DOCTYPE> element followed by the document type definition.
